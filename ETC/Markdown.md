# **Markdown**
>마크다운(Markdown)은 쓰고 읽기가 간편한 일반 텍스트 경향의 마크업 언어이다.
- 일반 텍스트로 서식이 있는 문서를 작성할 때 사용한다.
- 확장자명은 .md이다.
- 깃허브 내에서 .md파일을 편집할 수도 있으나, 별도의 편집기를 활용한 후 push를 하는 방법도 있다.
  
<br>
<p></p>

# **마크다운 문법**

## 1. 제목
- #을 사용하는 방법

```
# 제목을 입력
###### 최대 6개까지 가능하며, 많을 수록 폰트의 크기가 작아진다
```

- '==='를 사용하는 방법

```
===
제목
---
부제목
```

<br>

## 2. 목록
- 순서가 없는 목록

```
- 순서가 없는 목록은
* '-' 혹은 '*'를 앞에 입력해 작성 가능하다
```

- 순서가 있는 목록

```
1. 순서가 있는 목록은
2. '1.' 처럼 숫자와 점을 입력해 작성 가능하다
```

<br>

## 3. 코드 삽입
- '```'를 사용해 코드 삽입

<pre>
```
코드...
```
</pre>
>git에서는 첫 번째 '```' 뒤에 사용한 언어를 작성하여 문법을 강조할 수도 있다.

- 'pre' 태그를 사용해 코드 삽입

```
<pre>
<code>
코드...
</code>
</pre>
```

<br>

## 4. 인용 문구 (BlockQuote)

```
> 인용 문구
>> 서브 인용문구
```
> 인용 문구
>> 인용문구
